void generate_all_moves ( const Position & pos,
S_MOVELIST & list )

Generate all legal moves for the current position.

Main entry point for move generation during search. Uses optimized piece-specific modules to generate moves efficiently, with each module targeting specific performance bottlenecks identified through profiling.

Parameters
posCurrent chess position
listMove list to populate with generated moves

The function leverages:

  • PawnOptimizations: Handles 20.3% of generation overhead
  • SlidingPieceOptimizations: Handles 45%+ of generation overhead
  • Knight/King optimizations: Template-based fast generation
  • Direct piece list iteration: Fastest traversal method
