Detailed Description

Huginn Chess Engine - King Move Generation Optimizations

This module implements optimized king move generation by separating ordinary king moves from castling logic, based on profiling results showing king moves consume 19.1% of move generation time.

Optimization Strategy:

  1. Pre-check castling viability to avoid expensive Huginn::SqAttacked calls
  2. Separate ordinary king moves from castling moves
  3. Use early returns for impossible castling scenarios
  4. Cache-friendly data structures for quick castling validation
