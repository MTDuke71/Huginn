Detailed Description

Implementation of cross-platform input detection for chess engine responsiveness.

This file implements the platform-specific details for non-blocking input checking. It handles the fundamental differences between Windows and Unix-like systems in their approaches to console input detection, providing a unified interface for the chess engine's search interrupt capabilities.

Platform-Specific Implementations:

  • Windows: Uses _kbhit() from conio.h for immediate keyboard detection
  • Unix/Linux: Uses select() system call with zero timeout for POSIX compliance

Integration Notes: These functions are designed for periodic calling from search loops to maintain engine responsiveness without significant performance impact. The implementation prioritizes minimal overhead and maximum compatibility across platforms.
