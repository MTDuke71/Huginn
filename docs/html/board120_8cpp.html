Detailed Description

Implementation of 120-square coordinate system utilities.

Contains non-performance-critical utility functions for the 120-square mailbox coordinate system, including algebraic notation parsing and formatting. These functions are moved from headers to reduce compilation overhead while maintaining functionality for I/O operations and debugging.

Functions

  • from_algebraic(): Parse square names like "e4", "a1" to 120-square indices
  • to_algebraic(): Format 120-square indices to readable notation
  • Input Validation: Robust error handling for invalid coordinates
